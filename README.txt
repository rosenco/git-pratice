結帳：
1.可列出購買清單、總金額
2.結帳時會清空購物車

建立帳號：(尚未有token)
帳號密碼可以寫入結帳購物車的table


倉庫：
1.可以建立庫存商品、庫存數量、商品單價
2.可以刪除庫存商品
3.可以將庫存商品的數量、單價異動(但不直接影響已經放入購物車的商品)
4.可以查詢庫存商品列表

購物車：
1.只能購買倉庫有的商品，可以將商品放入購物車；並會先檢查庫存是否足夠，若不足夠則無法購買，最後會計算該商品總量的總金額
2.可以刪除購物車商品，並且刪除的數量將會返回倉庫中
3.可以修改購買的商品數量，並會先計算修改後的數量是否比庫存大，若比庫存大則無法修改
若比庫存小將可修改購物車內容，並且也會重新計算總金額
4.可以查詢購物車內容
